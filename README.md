# MISP-2.5-Install-Script
This file is a Bash installation script designed to automate the deployment of MISP (Malware Information Sharing Platform) version 2.5 on a fresh Ubuntu 24.04 LTS server.

The script installs all necessary dependencies, configures essential components, and prepares the environment for MISP operation, following the principles outlined in the official MISP installation documentation and community-maintained scripts.

Key Features and Functions

Purpose and Scope

Automates the end-to-end installation of MISP 2.5.

Intended for new Ubuntu 24.04 LTS systems.

Sets up both the application stack and its database backend.

Sources and Credits

Draws from:

Official MISP installation guides: https://misp.github.io/MISP

Usage: sudo ./mispinstall.sh

Ensure that you make the script executable: sudo chmod +x mispinstall.sh
